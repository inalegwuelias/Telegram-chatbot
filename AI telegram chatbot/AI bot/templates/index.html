Help & Instructions
Bot Features:
  • AI Assistant: Responds to questions and queries in Telegram
  • Content Moderation: Automatically moderates group chat messages
  • Rule Enforcement: Handles violations according to defined rules
Admin Commands:
  • /ban @username [reason] - Ban user permanently
  • /tempban @username [hours] [reason] - Ban for specific hours
  • /unban @username - Unban a user
  • /warn @username [reason] - Add violation to user
  • /unwarn @username - Remove one violation
  • /reset @username - Reset all violations
  • /stats - Show group statistics
