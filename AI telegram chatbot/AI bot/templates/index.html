Help & Commands
User Commands (in Telegram)
  • /start — Welcome message
  • /help — Show all commands
  • /rules — Display group rules
  • /violations — Check your violation count
Admin Commands
  • /ban @username [reason] — Permanently ban a user
  • /tempban @username [hours] [reason] — Temporarily ban a user
  • /unban @username — Unban a user
  • /warn @username [reason] — Add a violation
  • /unwarn @username — Remove a violation
  • /reset @username — Reset all violations
  • /stats — Show group statistics
